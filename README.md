# UniV Crutch (WIP)

Приложение расширенных настроек моделей Changan Uni-V на прошивке глобального экспорта (см. [Совместимость ГУ](#совместимость-с-гу))   

Реализованные настройки с последнего релиза:
+ Управление заслонками выхлопа через функциональную кнопку на руле (для спорт комплектаций)
+ Автоматическое отключение Start&Stop при запуске

## Установка

1. Скачиваем/передаем последний актуальный APK из [Releases](https://github.com/DEFLIK/univ-swc281o-crutch/releases) на ГУ авто любым доступным способом (мессенджер, ссылка на скачивание, adb)
2. Открываем APK на ГУ, нажимаем "Установить"
   + ⚠️ Внимание! Если при первичной установке система предложит обновить/заменить существующее приложение, 
   либо возникнет конфликт подписи - имеется риск перезаписи критичного системного приложения. 
   В таком случае следует считать приложение несовместимым с ГУ и отменить установку
3. Переходим в инженерное меню для отображения иконки приложения
   + Открываем приложение "Телефон"
   + Набираем ```*#*#888```, жмем звонок
   + Вводим пароль ```369875```
   + Открывается инженерное меню -> выбираем пункт ```打开系统设置原始界面``` (настройки Android) -> "Приложения и уведомления" -> "Launcher" -> "Хранилище" -> "Очистить кэш"
   + Перезагружаем ГУ зажатием кнопки "-" (минус) на руле на 8 секунд, по окончании отпускаем кнопку
4.  В конце общего списка приложений находим нашу программу, запускаем нажатием по иконке
5.  Действуем дальше по инструкции из приложения

## Совместимость с ГУ

Приложение создавалось и тестировалось на прошивке ГУ для глобального рынка (в частности РФ) в комплектации Luxe 2024 года выпуска:
> Версия FOTA: SW:B.8.7
> 
> Версия MCU: SW:D.7.6JB
> 
> Версия ОС: SWC.281O.00.0012_20240604_Car_Y
> 
> Версия Screen: -

Есть вероятность работоспособности приложения (части настроек) на комплектации Tech (рынок РФ), но гипотеза не проверена

Для дилерских комплектаций китайского рынка (DLX, LX и т.д.), а также комплектаций параллельного импорта приложение имеет риск несовместимости
